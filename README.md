# PulseTime Pro | Premium Time Management Suite

PulseTime Pro is a premium SaaS-style multi-tool time management platform designed for professionals. It features a modern, clean interface with glassmorphism design and robust timing logic.

## 🌐 Live Demo
[https://pulsetime-pro.github.io/pulsetime-pro/](https://pulsetime-pro.github.io/pulsetime-pro/)

## ✨ Features

- **Stopwatch**: Start, pause, resume, reset, and millisecond precision with lap tracking.
- **Countdown Timer**: Custom hours/minutes/seconds with alarm notifications.
- **Sleep Timer**: Specialized timer with simulated audio fade-out effect.
- **Multi-Timer System**: Run multiple independent named timers simultaneously.
- **Pomodoro Timer**: Customizable work and break intervals (25/5/15 default).
- **Alarm System**: Set multiple alarms with custom labels and selectable sounds.
- **Theme Support**: Premium Light and Dark modes with glassmorphism effects.
- **Persistence**: Saves your theme, settings, multi-timers, and alarms to LocalStorage.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **SEO Optimized**: Complete meta tags, Open Graph, Twitter Cards, and JSON-LD.

## 🚀 Tech Stack

- **Frontend**: HTML5, CSS3 (Custom Properties, Flexbox, Grid), JavaScript (ES6+)
- **Icons**: Font Awesome 6
- **Typography**: Inter (Google Fonts)
- **Deployment**: GitHub Pages

## 🛠️ Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/pulsetime-pro/pulsetime-pro.git
   ```
2. Open `index.html` in any modern web browser.
3. No build step or backend required!

## 📦 Deployment to GitHub Pages

1. Create a new repository on GitHub named `pulsetime-pro`.
2. Initialize git and push the code:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: PulseTime Pro v1.0"
   git branch -M main
   git remote add origin https://github.com/pulsetime-pro/pulsetime-pro.git
   git push -u origin main
   ```
3. Go to Repository **Settings** > **Pages**.
4. Select the `main` branch and `/root` folder, then click **Save**.
5. Your site will be live at `https://pulsetime-pro.github.io/pulsetime-pro/`.
